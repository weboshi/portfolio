PolitSearch

React / Google Civics API / ProPublica API / OpenSecrets API

A react app that lets users enter their address to find their representatives. Returns representatives' political affiliation, office and social media. For federal level representatives, users can also see how much campaign funds they received in their last election.

Tokyo Music Search

React / Cheerio

A react app that lets users search Tokyo's most popular music venues for artists that are performing. The search utilizes Cheerio to do a live scrape of the venue's schedules, ensuring that performance info is up to date. If an artist is performing, the month and schedule link are returned to the user.

TechCheck

React / Role: Front End / AWS / Logo

A commerce site that serves as a marketplace for used electronic hardware, PC components and accessories. Users can register and post listings and purchase items. Users can search the database for listings and can organize by category.

Moochify

Role: Front End

A marketplace website that connects owners and renters. Allows owners to list their items for rent, with condition, price and availability. Renters can search for items and connect with the buyer.

JCON

React

A site built to showcase a fictional convention centered on Japanese pop culture. Shows design, css effects and layout.
